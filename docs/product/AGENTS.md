# Product Docs Contract

This folder owns the product vision, PRD, MVP scope, user flows, and UX expectations.

Update these docs when changing:

- Product positioning.
- Target users.
- User-facing flows.
- Mobile UX.
- Onboarding.
- Feature scope.
- MVP definition.
- Copy or conceptual model.

Do not put low-level implementation details here unless they affect the product experience.

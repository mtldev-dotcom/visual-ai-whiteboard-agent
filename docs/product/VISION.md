# Product Vision

## Product name placeholder

Visual AI Whiteboard Assistant.

## Core idea

The app is an AI assistant with a living canvas.

Users can chat with the assistant from web UI or Telegram. The assistant can answer normally, but it can also create and manipulate a visual workspace.

The whiteboard is not only for drawing. It is a place where the assistant can build structured boards, mini apps, notes, diagrams, task lists, reminders, and dashboards.

## Product promise

Talk to your assistant. It builds the workspace around your ideas.

## Main differentiator

Most whiteboard tools are manual. Most AI assistants are chat-only. This app combines:

- Chat assistant.
- Visual board.
- Assistant-controlled structured canvas.
- Prebuilt widgets.
- Sandboxed custom HTML widgets.
- Markdown-based assistant core files.
- Telegram control surface.
- Mobile-first UX.

## Product framing

Use this framing:

> A visual AI assistant workspace.

Avoid this framing:

> AI Miro clone.

The assistant is the product. The canvas is its visual hands.

## Ideal user experience

1. User asks for something in chat.
2. Assistant understands whether text or visual output is better.
3. Assistant creates or updates a board.
4. User can manually edit everything.
5. Assistant can explain, summarize, reorganize, or extend the board later.
6. User can capture quick notes/tasks from Telegram.
7. The board becomes a persistent, organized workspace.
